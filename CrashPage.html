hash₁ = hexString (输入的初始哈希)

inputBytes = TextEncoder.encode(previousHex)

hashₙ = SHA256(inputBytes)

hexStrₙ = bytesToHex(hashₙ)

value = int(hexStr[0:13], 16)

X = value / 2^52

multiplier = 99 / (1 - X)

result = round( max(100, multiplier) ) / 100 + "x"

· 我们预先通过加密算法生成一千万个随机数字，并按照倒序依次对应每一轮游戏——也就是说，第一轮的结果是这串数字的最后一个，第二轮是倒数第二个，以此类推； · 这意味着所有轮次的爆点数值在游戏开始前就已经确定，完全独立于玩家的消费行为，杜绝了任何形式的实时操纵； · 每一轮的爆点结果均可通过公开的哈希算法进行追溯与验证； · 任何支持 SHA256 验算的网站都可验证哈希真伪。
